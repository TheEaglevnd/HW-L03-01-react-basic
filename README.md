React Basics Practice
A set of beginner-level React exercises built as part of the Daneshkar Frontend Bootcamp, focused on core concepts like components, JSX, and basic app structure.

About
This repository contains several small React practice folders, each exploring a different fundamental concept of React development — from setting up components to working with JSX syntax and simple app structures.
